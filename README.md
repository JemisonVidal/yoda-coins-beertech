# yoda-coins
Yoda Coin project for beertech studies.

## Usage

### Commands

```sh
yarn install
```

#### Usage

```sh
yarn start
```

Or dev env

```sh
yarn start-client
```

#### Build project to production

```sh
yarn build
```
