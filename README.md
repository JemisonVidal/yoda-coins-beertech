# yoda-coins
Yoda Coin project for beertech studies.

## Usage

### Commands

```sh
yarn install
```

## Usage

### Dev

```sh
yarn dev
```

### Deploy
```sh
yarn start
```

#### Build project to production

```sh
yarn build
```
